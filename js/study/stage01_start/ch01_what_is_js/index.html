Ch01. 자바스크립트(JavaScript)란?

Hello, 자바스크립트!

웹 페이지에 동적인 기능을 추가하는 언어입니다.

자바스크립트의 특징

  • 인터프리터 언어 (한 줄씩 즉시 실행)
  • 동적 타입 (변수 타입이 실행 중에 결정)
  • 프론트엔드 + 백엔드 모두 사용 가능
  • HTML / CSS 와 함께 웹 개발의 3대 핵심 언어

이벤트 실습

아래 버튼을 클릭하면 자바스크립트가 DOM 을 변경합니다.

버튼을 클릭하면 여기가 바뀝니다.

자바스크립트 연결 방법

방법 1. 인라인 이벤트 핸들러

HTML 태그의 이벤트 속성에 직접 JS 코드를 작성합니다. 간단하지만 유지보수가 어려워 권장하지 않습니다.

방법 2. 내부 스크립트 <script>

HTML 파일 안에 <script> 태그를 사용해 JS 코드를 작성합니다.

<script> console.log('내부 스크립트 실행'); </script>

방법 3. 외부 파일 연결 (권장) ✅

별도의 .js 파일을 만들고 src 속성으로 불러옵니다. 가독성이 높고 캐싱·재사용이 가능해 권장하는 방식입니다.

<script src="script.js"></script>

async / defer 속성 (로딩 최적화)

<head> 에서 스크립트를 로드할 때 HTML 파싱을 차단하지 않도록 합니다.

<!-- defer: HTML 파싱 완료 후 순서대로 실행 (가장 권장) --> <script src="script.js" defer></script> <!-- async: 다운로드 완료 즉시 실행 (실행 순서 보장 안 됨) --> <script src="script.js" async></script>
