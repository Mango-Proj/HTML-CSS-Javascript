01 Window & BOM 기본 정보

페이지가 열리는 순간 브라우저와 화면 정보를 읽어옵니다. navigator, screen, window 객체에서 가져옵니다.

02 대화 상자 (Dialog)

alert · confirm · prompt는 브라우저가 제공하는 내장 팝업 창입니다. 실행되는 동안 나머지 코드 실행이 일시 정지됩니다.

alert()

확인 버튼만 있는 안내 창

confirm()

확인(true) / 취소(false) 반환

prompt()

텍스트 입력 → 문자열 반환

03 Location & 스크롤

location 객체는 현재 페이지의 URL 정보를 담고 있습니다. URL을 파싱하거나 다른 페이지로 이동할 때 사용합니다.

04 타이머 (Timer)

setTimeout은 지정한 시간 후 딱 한 번 실행합니다. setInterval은 지정한 간격으로 반복 실행합니다. 두 함수 모두 비동기로 동작합니다.

setTimeout

3초 후 메시지를 표시합니다.

대기 중

setInterval — 스톱워치
