🌳 DOM 마스터 가이드

JavaScript로 HTML 요소를 선택하고, 수정하고, 생성하고, 삭제하는 방법을 배웁니다.

01 요소 선택 (Selection)

JavaScript가 HTML 요소를 조작하려면 먼저 해당 요소를 선택해야 합니다. 아래 버튼을 눌러 각 선택 방법의 차이를 확인해보세요.

id="box-target" 박스 (getElementById 대상)
class="highlight-me" 첫 번째 (querySelector 대상)
class="highlight-me" 두 번째 (querySelectorAll 대상)
class="highlight-me" 세 번째 (querySelectorAll 대상)

02 콘텐츠 조작

요소의 내용을 읽거나 바꿀 때 사용하는 세 가지 속성을 비교합니다. textContent, innerHTML, innerText는 각각 다르게 동작합니다.

원본 텍스트입니다. 숨겨진 텍스트 (display:none)

03 classList 조작

요소의 클래스를 추가·제거·토글하는 방법입니다. active 클래스가 있으면 박스 색이 바뀝니다.

classList 조작 대상 박스
현재 클래스: demo-box

04 속성 & 스타일 조작

getAttribute / setAttribute로 HTML 속성을 읽고 쓰거나, dataset으로 data-* 커스텀 속성을 읽을 수 있습니다. element.style로 CSS를 직접 변경할 수도 있습니다.

제목: DOM 기초 학습

링크: 원래 링크 텍스트

data-author / data-category 속성 보유

스타일 직접 변경 박스

05 노드 생성 & 삭제 (신호등)

createElement로 새 요소를 만들고, appendChild로 DOM에 붙이거나 removeChild로 제거할 수 있습니다.
