ch03 · 데이터 관리

불변성, 모듈화, localStorage / sessionStorage / Cookie를 직접 실행하며 확인하는 인터랙티브 예제입니다.

01 불변성 (Immutability)

불변성이란 데이터를 직접 수정하지 않고, 변경이 필요할 때 새로운 데이터를 생성하는 원칙입니다.
const는 재할당만 막습니다 — 배열·객체 내부 값은 여전히 변경될 수 있습니다.

📦 배열 (Array) 불변성

원본 배열 (originalArr) [1, 2, 3]
→
새 배열 (newArr)

🗂️ 객체 (Object) 불변성

원본 객체 (originalUser) { name: 'Alice', score: 100 }
→
새 객체 (newUser)

    02 모듈화 (Modularization)

    코드를 독립적인 파일(모듈)로 나누어 관리합니다.
    export로 기능을 내보내고 import로 가져옵니다.
    <script type="module">로 로드해야 모듈 문법이 동작합니다.

    theme.js
    export const THEME_KEY = 'user_theme';
    export function saveTheme(theme) { … }
    export function loadTheme() { … }
    import →
    main.js
    import { saveTheme, loadTheme, THEME_KEY }
      from './theme.js';
    saveTheme('dark'); // 저장
    loadTheme(); // 불러오기

    🎨 테마 영속화 데모 (localStorage 연동)

    헤더의 다크 모드 버튼을 클릭하면 theme.js의 saveTheme()이 호출되어
    localStorage['user_theme']에 선택한 테마가 저장됩니다. 페이지를 새로고침해도 유지됩니다.

      03 localStorage

      탭·창을 닫아도 유지되는 브라우저 저장소입니다.
      같은 도메인의 모든 탭에서 공유되며, 명시적으로 삭제하기 전까지 데이터가 보존됩니다.

      저장된 항목이 없습니다.

        04 sessionStorage

        탭(창)을 닫으면 자동 삭제되는 임시 저장소입니다.
        각 탭은 독립된 sessionStorage를 가지며, 같은 도메인이라도 탭 간 데이터가 공유되지 않습니다.

        특성localStoragesessionStorage
        유지 기간영구 (명시적 삭제 전)탭·창 닫힘 시 삭제
        탭 간 공유✅ 공유됨❌ 탭마다 독립
        주요 용도테마, 자동 로그인, 설정임시 입력, 일회성 데이터
        APIsetItem · getItem · removeItem · clear (동일)

        저장된 항목이 없습니다.

          05 Cookie

          쿠키는 서버 요청 시 HTTP 헤더에 자동으로 포함되는 작은 데이터입니다.
          document.cookie로 읽고 쓰며, expires 속성으로 만료 시간을 지정합니다.
          설정·삭제 시에는 반드시 path=/를 지정해야 같은 경로에서 접근할 수 있습니다.

          쿠키 저장

          쿠키 삭제

          저장된 쿠키가 없습니다.

          cookie_example.js — 기본 사용 예시

          // 쿠키 저장: expires 로 만료일 지정, path 로 유효 범위 설정
document.cookie = "visitor=JohnDoe; expires=Fri, 31 Dec 2026 12:00:00 UTC; path=/";

// 쿠키 읽기: 모든 쿠키가 "key=value; key2=value2" 형태의 문자열로 반환
console.log("현재 쿠키 목록:", document.cookie);

// ⚠️ 쿠키 삭제: expires를 과거 시간으로 설정
document.cookie = "visitor=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/";
