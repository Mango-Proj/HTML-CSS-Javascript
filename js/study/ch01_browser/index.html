01 DocumentFragment & 이벤트 위임

DocumentFragment는 메모리 상의 가벼운 DOM 컨테이너입니다.
여러 노드를 fragment에 모은 뒤 appendChild(fragment) 한 번으로 실제 DOM에 반영하면 리플로우가 1회만 발생합니다.

이벤트 위임: 리스너를 <ul> 부모에 하나만 달아도 동적으로 추가된 <li> 클릭을 처리할 수 있습니다.

  • 항목 A
  • 항목 B
  • 항목 C

li를 클릭하면 토글됩니다. 추가 버튼은 리플로우를 1회만 유발합니다.

      05 async vs defer 스크립트 로딩

      HTML 파싱을 막지 않고 스크립트를 로드하는 두 가지 방법의 차이입니다.

      <script>
      파싱
      ⛔ 다운로드+실행
      파싱 재개
      async
      파싱 (계속)
      ⚡ 즉시 실행
      파싱 재개
      defer
      파싱 완료까지 계속 ✅
      📌 파싱 후 실행
      ❌ 비권장 (파싱 차단)
      <head>
  <script src="app.js"></script>
</head>
      ⚠️ async (순서 불보장)
      <head>
  <script async src="analytics.js">
  </script>
</head>
      ✅ defer (권장)
      <head>
  <script defer src="app.js">
  </script>
</head>
