# CSS 단위와 색상 (Units & Colors)

CSS에서 크기를 표현하는 단위와 색상을 지정하는 방식을 정리합니다.

---

## 1. CSS 크기 단위 (Units)

단위는 크게 **절대 단위**와 **상대 단위**로 나뉩니다.

### 절대 단위 (Absolute)

| 단위 | 설명 |
|------|------|
| `px` | 화소(Pixel) 하나 기준. 디바이스 화면과 무관하게 크기가 고정됨 |

### 상대 단위 (Relative)

주변 환경(부모 요소, 루트 설정 등)에 따라 크기가 유동적으로 변합니다.

| 단위 | 기준점 | 특징 |
|------|--------|------|
| `em` | 부모 요소의 `font-size` | 중첩될수록 크기가 누적되어 예측이 어려울 수 있음 |
| `rem` | `<html>`(루트)의 `font-size` | 기준이 항상 고정되어 계산이 명확함 **(권장)** |
| `%` | 부모 요소의 크기 | 레이아웃의 너비·높이를 유동적으로 잡을 때 사용 |

```css
html { font-size: 16px; }

.parent { font-size: 20px; }

.child-em  { font-size: 1.5em;  } /* 부모 20px × 1.5 = 30px */
.child-rem { font-size: 1.5rem; } /* 루트 16px × 1.5 = 24px */
```

### 뷰포트 단위 (Viewport)

사용자가 보고 있는 브라우저 화면 크기를 기준으로 합니다.

| 단위 | 설명 |
|------|------|
| `vw` | 화면 너비의 1% (100vw = 화면 전체 너비) |
| `vh` | 화면 높이의 1% (100vh = 화면 전체 높이) |

```css
.banner {
  width: 100vw;  /* 브라우저 너비 전체 */
  height: 20vh;  /* 브라우저 높이의 20% */
}
```

## 2. 색상 표현 방식 (Color Models)

### RGB / RGBA

빛의 3원색(Red, Green, Blue)을 `0`~`255` 숫자로 조합합니다.
`RGBA`는 여기에 **Alpha(투명도)** `0`(투명) ~ `1`(불투명)을 추가합니다.

```css
color: rgb(0, 123, 255);           /* 파란색 */
color: rgba(0, 123, 255, 0.7);     /* 파란색, 70% 불투명 */
```

### Hex Code (16진수)

`#RRGGBB` 형식으로 각 색상 강도를 `00`~`ff` 16진수로 표현합니다.
웹 디자인에서 가장 많이 사용되는 방식입니다.

```css
color: #ffffff;  /* 흰색 */
color: #000000;  /* 검은색 */
color: #28a745;  /* 초록색 */
```

### opacity vs rgba 비교

두 방식 모두 투명도를 조절하지만 적용 범위가 다릅니다.

| 방식 | 투명도 적용 범위 | 사용 예 |
|------|-----------------|---------|
| `rgba(r, g, b, a)` | **배경색만** 투명하게 → 텍스트는 유지 | 반투명 배경 박스 |
| `opacity: 0~1` | 요소 **전체**(배경 + 텍스트 + 자식) 투명 | 요소 전체 페이드 효과 |

```css
/* 배경만 반투명, 글자는 선명하게 유지 */
.box-rgba    { background-color: rgba(0, 123, 255, 0.5); }

/* 박스 전체(배경 + 글자)가 반투명 */
.box-opacity { background-color: #28a745; opacity: 0.5; }
```
