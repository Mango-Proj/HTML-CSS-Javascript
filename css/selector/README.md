# CSS 선택자 (Selector)

CSS 선택자는 스타일을 적용할 HTML 요소를 지정하는 패턴입니다.

---

## 선택자 종류

### 1. 전체 선택자 `*`

문서 내 **모든 요소**를 선택합니다. 주로 기본 여백 초기화에 사용합니다.

```css
* {
  margin: 0;
  padding: 0;
}
```

---

### 2. 아이디 선택자 `#id`

`id` 속성 값이 일치하는 **단 하나의 요소**를 선택합니다. 페이지 내 고유해야 합니다.

```css
#main-title {
  font-size: 32px;
  color: navy;
}
```

---

### 3. 클래스 선택자 `.class`

`class` 속성 값이 일치하는 **모든 요소**를 선택합니다. 여러 요소에 반복 사용 가능합니다.

```css
.text {
  color: gray;
}
```

---

### 4. 교집합 선택자 `.class1.class2`

두 클래스를 **동시에 가진 요소**만 선택합니다. 선택자 사이에 공백 없이 붙여 씁니다.

```css
/* class="text blue" 처럼 두 클래스를 모두 가진 요소 */
.text.blue {
  color: blue;
}
```

---

### 5. 자식 선택자 `A > B`

A의 **직계 자식** B만 선택합니다. 손자 이하는 포함되지 않습니다.

```css
/* .container 바로 아래 p만 선택 */
.container > p {
  font-weight: bold;
}
```

---

### 6. 자손 선택자 `A B`

A 안에 있는 **모든 깊이의 B**를 선택합니다.

```css
/* .container 안의 모든 p (직계 + 손자 이하 포함) */
.container p {
  border-bottom: 1px solid silver;
}
```

> **자식 vs 자손 비교**
> | 선택자 | 범위 |
> |--------|------|
> | `A > B` | A의 직계 자식 B만 |
> | `A B` | A 안의 모든 B (깊이 무관) |

---

### 7. 인접 형제 선택자 `A + B`

A 바로 **다음에 오는 형제** B 하나만 선택합니다.

```css
/* 첫 번째 span 뒤에 오는 span에만 적용 */
span + span {
  margin-left: 10px;
}
```

---

### 8. 일반 형제 선택자 `A ~ B`

A 이후에 오는 **모든 형제** B를 선택합니다.

```css
span ~ span {
  color: darkgray;
}
```

---

### 9. 그룹 선택자 `A, B`

여러 선택자를 `,`로 묶어 **동일한 스타일을 일괄 적용**합니다.

```css
h1, span {
  font-style: italic;
}
```

---

## 선택자 우선순위 (Specificity)

같은 요소에 여러 스타일이 충돌할 경우, 아래 순서로 우선 적용됩니다.

```
인라인 스타일  >  #아이디  >  .클래스  >  태그
```

| 선택자 | 우선순위 점수 |
|--------|-------------|
| 인라인 `style=""` | 1000 |
| `#id` | 100 |
| `.class` | 10 |
| `태그` | 1 |
