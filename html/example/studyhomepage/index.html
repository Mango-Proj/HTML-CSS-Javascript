스터디 소개

Mango Dev Study가 무엇인지 알아보세요.

우리는 누구인가요?

프론트엔드 개발자를 꿈꾸는 사람들이 모인 스터디입니다.
매주 목요일 저녁 8시, 온라인으로 함께 공부합니다.

스터디 방식은 발표 + 코드 리뷰이며, "배운 것을 가르치는 것이 최고의 학습"이라는 철학을 따릅니다.

활동 방식

우리의 모토는 완벽한 코드보다 동작하는 코드입니다.
매주 각자 공부한 내용을 발표하고 서로 피드백합니다.

📖 핵심 개념 사전

시맨틱 태그
태그 이름 자체가 의미를 갖는 HTML 요소. SEO와 접근성에 유리.
블록 요소
새 줄에서 시작하며 가로 전체를 차지하는 요소. (div, p, h1 등)
인라인 요소
줄 흐름 안에 배치되며 콘텐츠 크기만큼만 차지. (span, a, img 등)
폼 (Form)
사용자 입력을 받아 서버로 전송하는 컨테이너. action / method 속성 필수.
