표와 목록 태그를 복합적으로 사용하여 데이터를 체계적으로 보여주는 프로젝트입니다.

포함될 내용:

<nav>: 커피, 에이드, 디저트 등 카테고리 이동 링크(앵커 태그).

<section>: 각 메뉴 카테고리별 구역 나누기.

<table>: 메뉴 이름, 특징, 가격을 표 형식으로 정리 (<thead>, <tbody> 활용).

<dl>: 특정 시그니처 메뉴에 대한 상세 설명(용어 정의).

<video>: 카페의 분위기를 보여주는 배경 영상 삽입.

학습 포인트: 복잡한 데이터를 표(table)와 리스트(ul, dl)로 구조화하기.
