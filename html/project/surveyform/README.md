사용자로부터 다양한 데이터를 입력받는 과정을 완벽하게 이해할 수 있는 프로젝트입니다.

포함될 내용:

<form>: post 방식을 사용한 기본 틀.

다양한 <input> 타입: 이름(text), 이메일(email), 비밀번호(password), 나이(number).

선택 요소:

radio: 성별 또는 만족도 선택.

checkbox: 관심 분야 다중 선택.

<select>: 거주 지역 선택.

<textarea>: 건의 사항이나 상세 의견 입력.

학습 포인트: label과 input의 연결, 다양한 type 속성 및 required 같은 유효성 검사 속성 활용.
